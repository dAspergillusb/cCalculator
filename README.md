# cCalculator

## Project Name
The program is called cCalculator.

## Description
This calculator was created for educational purposes to learn Python programming and understand how different program modules interact with each other.
Currently, it features a classic calculator, a numeral system converter (up to base-36), an arithmetic calculator for operations between different numeral systems, a function graphing tool, and a derivative calculator that automatically plots the derivative's graph.

## Installation
The program requires Python version 3.12 or higher. To run correctly, the following additional libraries are required: `dearpygui`, `sympy`, `requests`.
There is also a standalone binary file for Linux systems (located in the `standalone` folder of this project) which does not require installation.
